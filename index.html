World Clock Project

This world clock app is a web-based application that allows users to view the current time in different cities around the world. Built with HTML, CSS, and JavaScript, the app provides a clean and intuitive interface for users to easily navigate and view time zones. By leveraging Moment.js, the app can accurately display the current time in real-time based on the user's device clock and location.

Weather Project

This weather app is a web application that uses HTML, CSS, and JavaScript to provide real-time weather information to users. By integrating with the SheCodes Weather API, the app can fetch data on temperature, humidity, wind speed, and more for any location around the world. Users can search for their city or allow the app to access their location and receive personalized weather updates.
